# Mi5 Headphone Volume Fix

This mod loads the sound config of MIUI china ROM systemlessly to have
better headphone volume on Mi5 Global ROM BETA\STABLE
Following file is loaded systemlessly
/system/etc/mixer_paths_tasha.xml

to ensure if it has replaced correctly you can verify it by 
looking for my following comments in the file

<!-- Loaded via Magisk Mi5 Headphone volume fix module                      -->

This module has tested on Mi5 Global beta 7.9.7
